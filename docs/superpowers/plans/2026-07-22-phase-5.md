# Phase 5 — Advanced AI + Desktop Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Production AI features (offline fallback, alert explanations, cost tracking, dataset quality), advanced OCR (multi-region, benchmarks, chunking), desktop polish (native notifications, portable mode, crash reporting, code signing, CI builds).

**Architecture:** Backend gets Ollama integration in `engines/ai_insights_engine.py`, alert explanation endpoint, token cost tracking, dataset quality endpoint, multi-region OCR fields, OCR benchmark suite, video chunking with resume. Desktop gets Sentry crash reporting, native Tauri notifications, portable mode via CLI flag, code signing config, GitHub Actions CI pipeline.

**Tech Stack:** Python 3.12, FastAPI, SQLite (WAL), Ollama (local HTTP API), Sentry (Tauri + React), Tauri v1, Pydantic v2, PIL/Pillow (synthetic OCR frames), FFmpeg (segmentation), GitHub Actions
