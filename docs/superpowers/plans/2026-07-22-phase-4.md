# Phase 4 — AI Intelligence + Distribution Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add event validation, structured AI outputs, real-time job progress, frame annotation export, prompt versioning, and desktop dependency bundling/encryption.

**Architecture:** Backend gets `engines/event_validator.py` for balance/bet/win reconciliation, Pydantic models in `backend/schemas/ai.py` for structured AI outputs, and `engines/prompt_manager.py` for prompt versioning. Frontend `JobsMonitor.tsx` upgrades from HTTP polling to WebSocket. Desktop gets SQLCipher via `pysqlcipher3` and Tauri bundle config.

**Tech Stack:** Python 3.12, FastAPI, SQLite (WAL), Pydantic v2, Anthropic API (urllib), React 18, TanStack Query v5, WebSocket (raw), Tauri v1, pysqlcipher3

- [ ] **Step 3:** Add `prompt_versions` table to `database/db.py` V8 schema:
  ```sql
  CREATE TABLE IF NOT EXISTS prompt_versions (
      id INTEGER PRIMARY KEY AUTOINCREMENT,
      name TEXT NOT NULL,
      version INTEGER NOT NULL,
      system_prompt TEXT NOT NULL,
      model TEXT DEFAULT 'claude-sonnet-4-6',
      temperature REAL DEFAULT 1.0,
      max_tokens INTEGER DEFAULT 1024,
      is_active INTEGER DEFAULT 0,
      created_at TEXT DEFAULT (datetime('now')),
      UNIQUE(name, version)
  );
  CREATE TABLE IF NOT EXISTS ab_results (
      id INTEGER PRIMARY KEY AUTOINCREMENT,
      session_id INTEGER NOT NULL,
      prompt_a_id INTEGER NOT NULL,
      prompt_b_id INTEGER NOT NULL,
      winner TEXT,
      metrics TEXT,
      created_at TEXT DEFAULT (datetime('now'))
  );
  ```
